# Epic 3: Admin

**Objectif :** Permettre à l'admin de gérer le contenu du site (animaux, infos SPA).

**Stories :**
- [3.1 Authentification](./story-3.1-authentication.md)
- [3.2 CRUD Animaux](./story-3.2-crud-animals.md)
- [3.3 Upload Photos](./story-3.3-photo-upload.md)
- [3.4 Gestion statut animal](./story-3.4-animal-status.md)
- [3.5 Édition infos SPA](./story-3.5-edit-spa-info.md)

**Critère de complétion :** L'admin peut se connecter, ajouter/modifier/archiver des animaux, et mettre à jour les infos de la SPA.

**Dépendances :** Epic 1 et Epic 2 doivent être terminés.

**Note architecture :** Pas de back-office séparé. Les boutons admin apparaissent sur les pages publiques quand l'utilisateur est connecté.
